# Mtd.Infrastructure

[![.NET Build and Test](https://github.com/CUMTD/Mtd.Infrastructure/actions/workflows/build-test.yml/badge.svg)](https://github.com/CUMTD/Mtd.Infrastructure/actions/workflows/build-test.yml)
![GitHub Release](https://img.shields.io/github/v/release/CUMTD/Mtd.Infrastructure?include_prereleases&sort=semver&display_name=release)

Common infrastructure code for CUMTD projects.
This library contains implementations of cross-cutting concerns such as logging, configuration, and dependency injection.
Currently it only contains Entity Framework Core implementations however it may be expanded in the future.
